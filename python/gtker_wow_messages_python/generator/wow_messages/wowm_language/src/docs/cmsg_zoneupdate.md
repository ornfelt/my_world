# CMSG_ZONEUPDATE

## Client Version 1.12

Sent by the client whenever it reaches a new area.

The client does not send an accurate area. For example when going to Sen'jin Village, the client will send `DUROTAR` (0x0E) and not `SENJIN_VILLAGE` (0x16F).
